Vamos adicionar um background color no body de toda a página

Para isto, iremos utilizar o CSS inline apenas como exemplo. Na tag <body> iremos adicionar os seguintes itens:
style="background-color: cor selecionada/escolhida"

E para colocarmos mudanças no texto? Iremos utilizar na tag do texto os seguintes itens:
style="color: cor desejada"
OBS: A forma de utilização do CSS inline pode trazar problemas quando falamos de um grande website. Por quê?
Porque, adicionar regras CSS para cada elemento HTML consome muito tempo e faz a sua estrutura HTML ficar bagunçada. Estilizar múltiplos elementos pode afetar o tamanho da sua página o tempo para download.

Sobre alinhamento de texto:

Podemos utilizar:
style="text-align: tipo do alinhamento"
